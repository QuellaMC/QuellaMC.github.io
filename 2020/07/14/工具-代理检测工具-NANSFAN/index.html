[工具] 代理检测工具-NANSFAN

简介

这是一个代理检测工具，为什么我要发这个呢
因为它可以开启很多线程，速度快，支持类型全
50000代理开800线程10超时大约5分钟就好了
可以与上个帖子的代理爬取工具一起使用

原理

这就不用我多说了，原理与其他软件几乎一样
只不过速度快了很多，有颜色区分好坏代理
支持AUTO模式自动识别代理类型并分类
自测SOCKS4, SOCKS5单独无法检测，是个BUG
只能通过AUTO模式检测SOCKS4,5代理

使用截图

使用方法

  1. 双击run.bat
  2. 在跳出的窗口中选择你存放代理的文件夹
  3. 选择线程数量和超时时间，自测(800线程，10超时)
  4. 享受你的检查之旅吧！！！！！！
